## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/stern9/supermarket-boiler.git  or
   git clone git@github.com:stern9/supermarket-boiler.git
   ```
2. Install NPM packages
   ```sh
   npm install / yarn
   ```
3. Run both client and server
   ```sh
   npm run dev / yarn dev
   ```
